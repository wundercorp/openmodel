# Roadmap

## 0.1

- Establish independent repository and package publication.
- Stabilize tokens, package entry points, catalogue, and release automation.

## 0.2

- Split internal component implementation into independently owned modules.
- Add interaction tests for overlays and complex keyboard patterns.
- Add visual regression coverage for light, dark, and responsive states.

## 0.3

- Add framework integration guidance for Next.js, Remix, and React Router.
- Add optional CSS layering and theme-extension APIs.
- Publish a component contribution RFC template.

Roadmap items are directional and may change based on contributor feedback.
